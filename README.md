# 🚀 API & UI Test Automation Project

This project demonstrates automated testing using Java, Selenium, REST Assured, and Maven.

## 🧪 Features

- ✅ API Testing with REST Assured
- ✅ UI Testing with Selenium WebDriver
- ✅ Page Object Model (POM)
- ✅ Allure Reporting with Screenshots
- ✅ Parallel Test Execution
- ✅ CI Pipeline with GitHub Actions

---

## 🛠️ Tech Stack

- Java 21
- Maven
- Selenium WebDriver
- REST Assured
- JUnit 5
- Allure Reports
- GitHub Actions (CI)

---
